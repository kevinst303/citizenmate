# Targeted 'Weak Area' Quizzes

Provide highly personalized value that justifies premium tier upgrades by generating custom quizzes consisting *only* of questions the user has previously answered incorrectly or struggled with.

This will leverage the existing SRS engine, requiring a "Targeted Practice" mode on the dashboard that queries the user's quiz history and filters for low-accuracy topics.
