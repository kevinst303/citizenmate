## Project: CitizenMate

CitizenMate is a Next.js 16 civic-education SaaS targeting Australian citizenship exam prep. It has three main user flows:
- **Quiz flow**: `/practice/[testId]` — multiple choice questions with progress tracking
- **Study flow**: `/study` → `/study/[topicId]` — flashcard-style study with topic/chapter navigation
- **Dashboard**: `/dashboard` — user stats, readiness score, widgets (weather, currency, ABS data)

The app also has a marketing landing page at `/` with hero, features, how-it-works, social proof, pricing, and FAQ sections.

## Design System: Conseil
The entire v1.0 milestone was a design overhaul applying the conseil.pixfort.com consulting template. All design reference is in `citizenmate/docs/research/conseil/`.

Key tokens:
- Primary: `cm-teal` (#006d77)
- Secondary: `cm-purple` (#3d348b)
- Card: 15px radius, 1px #E9ECEF border, dual-layer box-shadow
- Fonts: Poppins (headings), Inter (body)
- Container: max-w-[1140px]

## Tech Stack
- Next.js 16 (has breaking changes — read `node_modules/next/dist/docs/` before any Next.js API work)
- TypeScript, Tailwind CSS, Framer Motion
- Recharts for dashboard charts
- Supabase for auth/data
