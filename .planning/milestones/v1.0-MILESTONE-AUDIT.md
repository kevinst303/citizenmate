## Critical Gaps

### GAP-1: `conseil-teal` CSS Token Undefined

**Severity:** Critical (visible rendering defect)
**Affects:** DASH-03, STUD-03

The token `conseil-teal` is used in `subpage-hero.tsx` (lines 21, 35, 36):
- `bg-conseil-teal`
- `from-conseil-teal/95`
- `to-conseil-teal/80`

This token is not defined in `globals.css` `@theme inline` block. The app defines `cm-teal` (#006d77) but not `conseil-teal`. The SubpageHero teal header band renders with no background color on `/dashboard` and `/study` pages.

**Fix (1 line in subpage-hero.tsx):**
Replace all `conseil-teal` references with `cm-teal`:
- `bg-conseil-teal` → `bg-cm-teal`
- `from-conseil-teal/95` → `from-cm-teal/95`
- `to-conseil-teal/80` → `to-cm-teal/80`
- `from-conseil-teal` → `from-cm-teal`

Or alternatively, add `--color-conseil-teal: #006d77` to the `@theme inline` block in `globals.css`.
